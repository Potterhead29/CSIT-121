Some form elements.

  1. Select element :It defines a drop-down list,gives you the option to choose elements from a list.
  2. The input element: It allows you to input something and get information about what you put in.
  3. The label element: It creates a caption for an item in a user interface.It is useful for screen-reader users.
  4. The multiple element: It allows the user to select more than one value from a list.
  5. The button element : It defines a clickable button.
